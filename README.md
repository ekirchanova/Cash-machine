Эта программа симулирует работу банкомата на этапе работы с деньгами.
Можно сделать три вещи:
1) положить деньги
2) снять деньги
3) посмотреть состояние банкомата

Рассмотри каждую подробнее.
1) Положить деньги
   Необходимо заполнить какое количество купюр каждого номинала хотите положить на счет и нажать ok
   Если все купюры удалось положить, то банкомат вернется к главному меню.
   Если не удалось положить какие-то купюры, то банкомат выведет сумму, которую не удалось положить
2)Снять деньги
  Необходимо ввести сумму для снятия и выбрать хотите или нет мелкими купюрами получить.
  Если необходимую сумму удалось снять, то банкомат вернется к главному меню.
   Если не удалось положить какую-то часть суммы, то банкомат выведет сумму, которую не удалось снять
3)Просмотр состояния
  Банкомат отобразит сумму, которая в нем хранится и количество купюр.
-----------------------------------
This program simulates the operation of an ATM at the stage of working with money.
There are three things it can do:
1) put money
2) withdraw money
3)show state

Consider each one in more detail.
1)Put money
   You need to fill out how many bills of each denomination you want to deposit into the account and click ok
   If all the bills have been deposited, the ATM will return to the main menu.
   If it was not possible to deposit any bills, the ATM will withdraw the amount that could not be deposited.
2)Withdraw money
  You need to enter the withdrawal amount and choose whether or not you want to receive small bills.
  If the required amount has been withdrawn, the ATM will return to the main menu.
   If it was not possible to deposit any part of the amount, the ATM will withdraw the amount that could not be withdrawn.
3)Show state
  The ATM will display the amount that is stored in it and the number of bills.
